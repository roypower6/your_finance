# **Your Finance**

> **💡 한국은행 Open API를 이용하여 만든 금융 정보, 사전 앱**

## **📱 소개**
한국은행 BOK Open API를 이용하여 실시간 금융 정보, 경제 관련 용어를 한국은행에서 직접 정리한 사전을 조회할 수 있는 어플리케이션입니다.
